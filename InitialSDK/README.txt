This directory contains the Initial Green Button SDK

Information on Green Button Initiative from SGIP:
http://collaborate.nist.gov/twiki-sggrid/bin/view/SmartGrid/GreenButtonInitiative

To purchase the NAESB ESPI Standard on which the Green Button file format is based, use the following link: http://www.naesb.org/misc/naesb_matl_order_espi_standards.pdf 

Files Included:
ESPIRender_xslt1.xslt - renders Green Button data in tabular form in a web browser
GETBatchSampleEUI.xml - sample Green Button data file
Green_Button_Initialtive_File_Format_Demo_20111014.docx - annotated description of file format rendering
Green_Button_Initiative_-_Requirements_v8_-_Final.docx - initial agreed upon requirements for CA IOU implementations of Green Button
LoadProfile.xlsm - Spreadsheet generator of load profile data (only Interval Data part of file)
requiredFieldsmatrix_v3.xlsx
